Grid Container

Para usar o grid layout precisamos definir o elemento "pai" com display grid ou inline-grid. O inline-grid torna o elemento um grid container, porém com comportamento inline.

grid-template-columns

Para definir o tamanho das colunas de cada elemento usamos o grid-template-column e o tamanho que cada coluna ocupara, podendo ser o tamanho em px (pixels) ou em fr (frames), px é valor absoluto e fr é valor relativo, a quantidade de colunas vai depender dos valores especificados pra essa propriedade, se for definido com 200px e 200px será criado duas colunas com 200px de tamanho, porém se mais "colunas" forem adicionadas nessa propriedade mais colunas terá o layout.

grid-template-columns: 200px 200px

Mesmo que tenha espaço na tela para serem criadas mais colunas será respeitado somente o tamanho de 200px.

grid-template-areas

grid-template-areas:
"logo nav nav nav"
"sidenav content advert categories"
"sidenav footer footer categories";

O grid-template-areas define os elementos como se fossem áreas, facilitando assim a estruturação da grid, poré para esse comportamento é preciso que sejam criadas as áreas no CSS, nesse caso temos as áreas, logo, nav, content, sidenav, advert, cotegories e footer, que na verdade são apenas classes, para que o CSS passe a enxerga-las como área, precisamos marcar essas classe com a propriedade grid-area e definir seus nome, por exemplo:
.logo {
grid-area: logo
}

.nav {
grid-area: nav
}

E assim por diante. E para criar a grid, realmente colocamos cada área na posição desejada, montando assim nossa grid. Nesse exemplo a logo ocupa a coluna 1 da linha 1, nav ocupa as coluna 2, 3 e 4 da linha 1, sidenav ocupa a coluna 1 da linha 2, content ocupa a coluna 2 da linha 1, advert ocupa a coluna 3 da linha dois, categories ocupa a coluna 4 da linha 2, sidenav ocupa a coluna 1 da linha 3, footer ocupa as colunas 2 e 3 da linha 3 e categories ocupa a coluna 4 da linha 3.

grid

Atalho geral para definri o grid: grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns e grid-auto-flow.

grid: 100px / 1fr 1fr
Gera uma linha com 100px de altura e 2 colunas com 1fr.

grid: 100px / auto-flow 100px 50px
Gera uma linha com 100px de altura. O grid-auto-flow é definido como column (pois está logo antes da definição das colunas). Ele também define o grid-auto-columns com 100px 50px

grid:
"logo nav nav" 50px
"sidenav content advert" 150px
"sidenav footer footer" 100px
/ 100px 1fr 50px;
Define as áreas, cria a priemira linha com 50px, a segunda com 150px e a terceira com 100px. Já as colunas são criadas com 100px a primeira, 1fr a segunda e 50px a terceira.
